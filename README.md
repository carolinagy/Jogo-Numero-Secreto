# Jogo do Número Secreto

Jogo interativo desenvolvido durante o curso de Lógica de Programação do **SENAC**, com algumas melhorias.

**[Jogar agora](https://carolinagy.github.io/Jogo-Numero-Secreto)**

---

##Como jogar
1. Pense em um número de **1 a 10**
2. Digite seu palpite e clique em **Tentar** (ou pressione Enter)
3. O jogo dá dicas se você está **quente** ou **frio**
4. Tente adivinhar com o menor número de tentativas possível!

---

##Funcionalidades

- Contador de tentativas
- Dicas de quente / morno / frio
- Histórico de palpites feitos
- Botão de reiniciar
- Suporte à tecla Enter
